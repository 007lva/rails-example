# Buildbox Rails Example

Use this as a guide on how to test Rails applications on Buildbox

## Databases

Set with `DATABASE_ADAPTER`

```
postgres
mysql
```

## Capybara Drivers

Set with `CAPYBARA_DRIVER`

```
firefox
chrome
poltergeist
webkit
```

## Installation (OSX)

```bash
brew install postgres mysql
brew install sphinx --pgsql --mysql
```
